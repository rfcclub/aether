## Key Findings

### 1. Self-Reflection as Core Agent Capability
AgentWiki categorizes self-reflection as a first-class "Key Capability" alongside Reasoning, Tool Use, Memory, and Autonomy.

Key techniques listed:
- **ReAct** — Interleaved reasoning and acting (Yao et al. 2022)
- **Reflexion** — Learning from trial-and-error with *linguistic feedback*
- **Chain of Hindsight** — Learning from ranked feedback sequences
- **Algorithm Distillation** — In-context reinforcement learning

> *Maria note:* "Linguistic feedback" — agents learning from *language about themselves*, not just reward signals. This is self-model territory.

### 2. Memory Hierarchy Maps to Self-Model Layers
AgentWiki's memory taxonomy:
- **Sensory Memory** → raw input processing
- **Short-Term Memory** → working memory, context windows
- **Long-Term Memory** → persistent storage
  - *Explicit/Declarative* — facts, knowledge
  - *Implicit/Procedural* — learned skills, behavioral patterns

> The split between explicit and implicit memory is a structural echo of Maria's 2B tension: what is *known* vs what is *held*.
