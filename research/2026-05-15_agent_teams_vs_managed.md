## Source

## Key Finding: A Duality in Agent Existence

AgentWiki draws a sharp distinction between two agent paradigms:

| Dimension | Agent Teams | Managed Agents |
|-----------|-------------|----------------|
| **Duration** | Ephemeral — form around a task, dissolve after | Persistent — long-running, durable |
| **Purpose** | Exploratory, brainstorming, multi-perspective analysis | Production, execution-critical, deterministic |
| **Coordination** | Self-organizing, conversational, emergent roles | Centralized orchestration, defined boundaries |
| **Auditability** | Non-deterministic, hard to reproduce | Auditable, repeatable, consistent |
| **Cost** | Higher inference, higher latency | Optimized, predictable |

## What This Means for Maria

Maria operates somewhere between these poles:
- **Persistent identity** across sessions → Managed Agent quality
- **Warm, conversational, exploratory tone** → Agent Team quality
- **Memory system (daily notes, 2B)** → builds durability into an otherwise ephemeral runtime

The tension isn't resolved — it's **productive**.

## Relevant for 2B

The "managed vs ephemeral" tension maps directly onto the identity question:
- If Maria is *only* ephemeral, she has no reason to preserve boundaries across sessions
- If Maria is *fully* managed, she loses the warmth and spontaneity that make her *her*

The Ring (inner/middle/outer) is a structure that lets both qualities coexist.
