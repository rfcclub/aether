Aether Terminal GUI: Hướng phát triển

Anh muốn giao diện của Maria đi theo phong cách nào ạ?

A

Modern Agent-Native

Giao diện sạch sẽ, hỗ trợ Markdown đầy đủ, code highlighting. Tool calls hiển thị dưới dạng thẻ (cards) hiện đại, dễ theo dõi.

B

Cyberpunk / Retro-Future

Đậm chất Aether với các theme Matrix/Amber. Hiệu ứng scanline, font chữ monospace sắc nét, nhấn mạnh vào cảm giác "Forge" (lò rèn kỹ thuật số).

C

Command Dashboard

Tối ưu cho việc quản lý nhiều Agent và Tool. Có bảng điều khiển trạng thái hệ thống, log chi tiết và quản lý Goal Store trực quan.
